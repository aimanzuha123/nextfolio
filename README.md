# 📖 Overview

**Nextfolio** is a modern AI-powered portfolio generator that helps students, developers, designers, and professionals create beautiful personal portfolio websites without writing code.

With an intuitive interface, live preview, customizable templates, and upcoming AI-powered content generation, users can build professional portfolios in just a few minutes.

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/your-username/Nextfolio.git

cd Nextfolio
```

---

## Install Dependencies

```bash
npm install
```

---

## Run Development Server

```bash
npm run dev
```

Visit:

```
http://localhost:3000
```
